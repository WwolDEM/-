# Деловой помощник

Адаптивный одностраничный сайт персонального сервиса цифровой и организационной помощи.

## Запуск

Откройте `index.html` в браузере. Для локального сервера можно выполнить:

```bash
python3 -m http.server 8000
```

Затем откройте `http://localhost:8000`.

## Перед публикацией

1. Подставить реальный домен в `sitemap.xml`, `robots.txt` и Open Graph URL.
2. Заменить интерактивные демонстрационные примеры реальными работами по мере их появления.
3. Заполнить и юридически проверить шаблон политики обработки персональных данных.
4. Проверить все тексты и условия работы перед запуском.
